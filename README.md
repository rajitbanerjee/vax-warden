# Vax Warden

COVID-19 vaccine registration system using Java, Spring Boot and React.js, developed for COMP47660 Secure Software Engineering.

## Requirements

- Docker
- JDK 11 and Apache Maven

## Getting Started

```
./run-services.sh
```
